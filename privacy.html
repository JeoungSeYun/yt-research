개인정보처리방침

떡상 리서치 (Tteoksang Research) · 시행일: 2026년 7월 12일

떡상 리서치(이하 "본 서비스")는 유튜브 크리에이터가 자신의 장르에서 성과가 좋은 영상과 채널을 검색·분석하도록 돕는 웹 기반 도구입니다. 본 서비스는 이용자의 개인정보를 최소한으로만 처리하며, 아래와 같이 그 방침을 안내합니다.

1. 수집·저장하는 정보

본 서비스는 백엔드 서버가 없는 정적 웹사이트로, 모든 처리는 이용자의 브라우저 안에서만 이루어집니다. 본 서비스 운영자는 어떠한 개인정보도 서버로 전송받거나 저장하지 않습니다.

  • YouTube Data API 키 — 이용자가 직접 입력한 키는 이용자 브라우저의 localStorage에만 저장되며, 외부로 전송되지 않습니다. (직접 Google API 호출에만 사용)
  • Google 로그인 정보 — Google 계정으로 로그인 시 이름·이메일·프로필 사진을 받아 화면에 표시하고 브라우저에만 저장합니다. 외부 서버로 전송하거나 제3자에게 제공하지 않습니다.
  • 이용자 저장 데이터 — 저장한 영상, 추적 채널, 검색 기록 등은 이용자 브라우저에만 저장되며 언제든 삭제할 수 있습니다.

2. YouTube API 서비스 이용

본 서비스는 YouTube API Services를 사용합니다. 이용자는 본 서비스를 사용함으로써 YouTube 서비스 약관에 동의하게 됩니다. 또한 Google의 개인정보 처리에 관해서는 Google 개인정보처리방침이 적용됩니다.

본 서비스는 YouTube에서 조회한 공개 데이터(영상·채널의 공개 통계)를 이용자에게 표시하는 용도로만 사용하며, 이를 별도로 저장하거나 제3자에게 판매·공유하지 않습니다.

3. 쿠키 및 추적

본 서비스는 광고·분석용 쿠키나 추적 기술을 사용하지 않습니다.

4. 데이터 삭제

모든 데이터는 이용자 브라우저에 저장되므로, 브라우저의 사이트 데이터 삭제 또는 서비스 내 로그아웃·삭제 기능을 통해 언제든지 완전히 제거할 수 있습니다.
