▶ YT 떡상 리서치 — 아무 장르나 검색해서 벤치마킹

🔑 API 키 무료 발급 방법 (5분)
  1. Google Cloud Console에서 새 프로젝트 만들기
  2. 상단 검색창에 YouTube Data API v3 검색 → 사용 버튼 클릭
  3. 왼쪽 사용자 인증 정보 → + 사용자 인증 정보 만들기 → API 키
  4. API 제한사항에서 YouTube Data API v3 선택 → 만들기
  5. 생성된 키를 복사해서 위 칸에 붙여넣기 (이 브라우저에만 저장, 서버 전송 없음)
무료 할당량: 하루 검색 약 10~30회 (자정 태평양시간 리셋)
