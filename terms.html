떡상 리서치 (Tteoksang Research) · 시행일: 2026년 7월 12일

1. 서비스 개요

떡상 리서치(이하 "본 서비스")는 유튜브 크리에이터가 자신의 장르에서 성과가 좋은 영상과 채널을 검색·분석하도록 돕는 무료 웹 도구입니다.

2. YouTube API 서비스

본 서비스는 YouTube API Services를 사용합니다. 이용자는 본 서비스를 사용함으로써 YouTube 서비스 약관 및 Google 개인정보처리방침에 동의합니다.

3. 이용자 책임

  • 이용자는 본인의 YouTube Data API 키를 직접 발급·입력하여 사용하며, 해당 키의 사용 및 할당량에 대한 책임은 이용자에게 있습니다.
  • 본 서비스를 불법적이거나 YouTube 약관에 위배되는 용도로 사용해서는 안 됩니다.

4. 면책

본 서비스는 공개된 YouTube 데이터를 있는 그대로 표시하며, 데이터의 정확성·완전성이나 이를 활용한 결과에 대해 보증하지 않습니다. 본 서비스 이용으로 발생하는 손해에 대해 운영자는 책임을 지지 않습니다.
